How to Create Encapsulation

STEP 1: Define a class

public class Person { // Private instance variables (fields) private String name; private int age;
// Constructor public Person(String name, int age) { this.name = name; this.age = age; }
// Public getter method for 'name' public String getName() { return name; }
// Public setter method for 'name' public void setName(String name) { this.name = name; }
// Public getter method for 'age' public int getAge() { return age; }
// Public setter method for 'age' public void setAge(int age) { this.age = age; } }

STEP 2: Create Objects and Access Data

public class Main { public static void main(String[] args) { // Create a Person object Person person1 = new Person("John", 30);
// Access data using getter methods System.out.println("Name: " + person1.getName()); System.out.println("Age: " + person1.getAge());
// Update data using setter methods person1.setName("Jane"); person1.setAge(25);
// Display updated data System.out.println("Updated Name: " + person1.getName()); System.out.println("Updated Age: " + person1.getAge()); } }
