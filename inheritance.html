How to Create Inheritance

Step 1: Identify the Base Class: Determine the class that will serve as the superclass, containing common properties and behaviors that you want to reuse in the subclass. Define the Superclass: Create the superclass with the desired fields, methods, and constructors.

public class Animal {
private String name;

public Animal(String name) {
this.name = name;
}

public void speak() {
System.out.println("I am an animal.");
}

public String getName() {
return name;
}
}


Step 2: Create the Subclass: Declare the subclass by using the extends keyword followed by the name of the superclass.

public class Dog extends Animal {
public Dog(String name) {
super(name); // Call superclass constructor
}

@Override
public void speak() {
System.out.println("Woof!");
}
}


Step 3:
public class Dog extends Animal {
public Dog(String name) {
super(name); // Call superclass constructor
}

@Override
public void speak() {
System.out.println("Woof!");
}
}


Override Methods (Optional): If you want to change the behavior of a method in the subclass, you can override it by using the @Override annotation. Access Superclass Members: Subclasses can access public or protected members of the superclass directly using the dot operator (.). Private members are not accessible.

STEP 4:
public class Main {
public static void main(String[] args) {
Dog dog = new Dog("Buddy");
System.out.println("Name: " + dog.getName()); // Output: Name: Buddy
dog.speak(); // Output: Woof!
}
}


Lastly, Run the Program: Compile and run the Java program to see the inheritance in action. You should observe that the subclass (Dog) inherits the properties and behaviors of the superclass (Animal) and can also have its own unique characteristics.
